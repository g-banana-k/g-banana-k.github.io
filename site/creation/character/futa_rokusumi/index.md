[ヘキサゴン・木口ハッピー連合バース廃校バース](/creation/world)の[亜麻仁床市](/creation/world/amaniyuka)に住んでいる。

彼が元の世界で転んだ拍子に裏返ったことで、彼の内側であったバース廃校バースが世界となり、元の世界は彼の内側に閉じ込められた。

# バター。
裏返った時に外に出てしまった精神。普段は体と同化しているが、外れることもある。

# ココア
常に装備している四次元ポケットのようなもの。四次元ポケットのような形をしているが、バース廃校バース内のどこかに繋がっているだけなため実態はどこでもドアに近い。裏返しても別に何も起こらない。

意思を持ち、勝手に物を出したりしまったりすることがある。
